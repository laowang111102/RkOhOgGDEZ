# 个人记账管理系统 python357

## 项目概述

个人记账管理系统是基于 Python Django 框架开发的，旨在帮助用户方便地管理个人财务，包括登录、个人信息管理、收支记录以及统计等功能。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：HTML

## 功能模块

### 用户模块

- 用户登录
- 个人信息修改

### 财务管理模块

- 收入记录
- 支出记录
- 收入记账管理
- 支出记账管理

### 财务统计模块

- 收入统计
- 支出统计

## 系统角色

- 普通用户：可以使用系统所有功能进行个人财务管理。

## 运行环境

- Python 环境推荐版本：3.6+
- Django 版本：2.2+
- MySQL 版本：5.7+

## 部署步骤

1. 环境搭建：准备 Python、Django 和 MySQL 的运行环境。
2. 数据库部署：根据提供的数据库文件初始化 MySQL 数据库。
3. 系统配置：配置 Django 项目的 settings.py，包括数据库连接等。
4. 运行项目：使用 Django 提供的命令启动开发服务器。

## 目录结构

```
project/
├── apps/

├── db/                      # 数据库相关文件目录

├── static/                  # 静态文件目录
├── templates/               # HTML 模板目录
├── manage.py                # Django 管理脚本
└── ...
```

## 核心亮点

- 基于成熟的 Python Django 框架，保证了系统稳定性与可扩展性。
- 提供详尽的记账管理功能，帮助用户清晰掌握财务状况。
- 简洁明了的统计界面，让用户直观了解财务状况变化。
